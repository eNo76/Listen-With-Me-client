# ʟɪꜱᴛᴇɴ ᴡɪᴛʜ ᴍᴇ (Qᴜᴀʀᴛᴇᴛᴛᴇ)

## 프로젝트 소개

**친구와 음악을 똑같이 들을 수 있다면 어떨까?** 에서 출발한 서비스 입니다. <br/>
내가 원하는 음악을 찾아 들을 수 있고, 친구의 코드를 입력해서 친구와 함께 들을 수도 있습니다.
